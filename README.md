# Homebrew tap for RapidKey

## Software

- [RapidKey](https://github.com/ivalkou/rapidkey) — keyboard-driven command palette for macOS

## Install

```bash
brew install --cask ivalkou/tap/rapidkey
```

Install a pinned version:

```bash
brew install --cask ivalkou/tap/rapidkey@0.1.0
```

## Install from a local cask file

```bash
brew install --cask ./Casks/rapidkey.rb
```
